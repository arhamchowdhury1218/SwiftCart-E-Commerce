#### 1) What is the difference between `null` and `undefined`?

Ans: The difference between null and undefined is that I am setting a variables value as null but undefined means I have not set any value for a variable.

#### 2) What is the use of the `map()` function in JavaScript? How is it different from `forEach()`?

Ans: Map() funtion means that it will itterate every value of an array and do an operation, then finally it will return a new array.
Map() function is different from forEach() because map() function returns a new array but forEach() do not return a new array.

#### 3) What is the difference between `==` and `===`?

Ans: `===` check the value along with the data type but `==` checks only the value. (Type cohersion)

#### 4) What is the significance of `async`/`await` in fetching API data?

Ans: The significance for `async`/ `await` is that when fetching an api data, it does not wait to execute the next code until the api data comes.

#### 5) Explain the concept of Scope in JavaScript (Global, Function, Block).

Ans: Global scope means if a variable is declared in a global scope you can access it from any where of your code file.

Function scope: it means the variable which you declared inside your function, that variable can only be access inside the function. You cannot access it outside the function scope.

Block scope: it means inside an if-else scope, the variable which is declared that variable can only be access inside that if-else scope.
